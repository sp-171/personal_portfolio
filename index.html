Graduate Student Admission System

A fully functional student admission system implemented in the terminal using linked lists. Features include creating, searching, sorting, and deleting students, merging domestic and international student lists, and updating CGPA and research scores. No graphical user interface; the program can be run using 'make' and './main' commands.
